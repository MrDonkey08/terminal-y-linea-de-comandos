| :-------: | :------------------------: | ------------------------------------------------------------------------- |
|     d     |       Archivo normal       |                                                                           |
|     l     |       Link simbólico       |                                                                           |
|     b     | Archivo de bloque especial | Archivo que manejea la información de los _bloques de datos_ como una USB |
